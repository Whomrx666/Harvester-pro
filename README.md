# Harvester-pro
![Harvester-pro preview](harvest.jpg)

## introduction
Harvester-pro is an OSINT (Open Source Intelligence) tool used to collect public information from the internet about a specific target (usually a domain or company).this is the pro version of the harvester.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ pkg install git -y
$ git clone https://github.com/Whomrx666/Harvester-pro.git 
$ cd Harvester-pro
$ python3 install.py

```
## Run manually
```
$ python3 harvest.py
```

## Instructions
- **First**: Install the tools according to the instructions above.
- **Second**: Run the main tools after installation.
- **Third**: Select one of the menus to run then enter the domain or IP address according to the instructions.
- **Last**: Tools will automatically do their job.
